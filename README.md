# Networking Tools

## Run from source

```bash
git clone https://github.com/tewarid/net-tools-electron
cd net-tools-electron
npm install
npm run start
```

## Package using electron-packager

```bash
git clone https://github.com/tewarid/net-tools-electron
cd net-tools-electron
npm install
npm install electron-packager -g
electron-packager . --all
```
